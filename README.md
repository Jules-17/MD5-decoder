# MD5-decoder
MD5-decoder est un outil pour retrouver un MDP à partir d'un Hash Md5 et d'un salt. 
Ce projet a été réalisé dans le cadre d’une PFMP en entreprise pour résoudre un CTF.

## Fonctionnalités

Cet outil permet de retrouver un mot de passe en essayant d'associer un mot d'une wordlist avec le salt, pour ensuite comparer le résultat avec le Hash et faire remonter le mot de passe si les 2 hash correspondent.
